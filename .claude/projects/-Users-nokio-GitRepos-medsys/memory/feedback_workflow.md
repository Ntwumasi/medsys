Always run migrations when needed, commit, and push changes after completing each task. Do not ask for permission.

**Why:** User wants autonomous workflow — asking slows things down and they trust the process.

**How to apply:** After every task: (1) run any new migrations, (2) verify build passes, (3) commit and push to main.
